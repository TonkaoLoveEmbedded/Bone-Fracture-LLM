# Project Overview
- โปรเจกต์นี้คือระบบวินิจฉัยภาพถ่าย x-ray และอธิบายผลด้วย LLM โดยในส่วนของfrontend สร้างด้วย React Typescript
- ทำแค่ในส่วน frontend ก่อน

## Project Structure
- `src/pages/` — 1 หน้า = 1 โฟลเดอร์ component ย่อยที่ใช้แค่หน้านั้นอยู่ในนี้
- `src/components/` — component ที่ใช้ซ้ำได้ทั้งเว็บ

## Tech Stack
- React + TypeScript + Vite

# Coding Rules
- ฝั่ง Frontend ใช้ TypeScript ทุกไฟล์
- ฝั่ง Backend ใช้  Javascript ได้
- เขียนโค้ดให้อ่านง่ายมากกว่าสั้นเกินไป
- ห้ามลบโค้ดเดิมถ้าไม่ใช่หน้าที่ของมัน
- ถามผมก่อนที่จะแก้ไขโค้ดเสมอ
